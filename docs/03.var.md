# Variable and Constant

## Variable

### TS

```ts
let b = "B";
b = "C";
```

### Go

```go
var one int = 1
var fourTwo float64 = 4.2
var fourTwoThree float32 = 4.23

var a = "A"
var a string = "A"

var b string
b = "B"
b = "C"

c := "C"
c = "D"
```

## Constant

### TS

```ts
const one: number = 1;
const fourTwo: number = 4.2;
const a = "A";
```

### Go

```go
const Pi = 3.14
```

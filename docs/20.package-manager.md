# Package manager

#### TS

init node project

```sh
npm init
```

install package

```sh
npm install <package-name>
```

remove package

```sh
npm uninstall <package-name>
```

#### Go

init go project

```sh
go mod init <module-name>
```

install pacakge

```sh
go get <package-name>
```

// remove pacakge

```sh
go mod tidy
```

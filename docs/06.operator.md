## Logical Operator

### TS

```ts
true && true; // true
true && false; // false
false && true; // false
false && false; // false

true || true; // true
true || false; // true
false || true; // true
false || false; // false

!true; // false
!false; // true
```

### Go

```go
true && true // true
true && false // false
false && true // false
false && false // false

true || true // true
true || false // true
false || true // true
false || false // false

!true // false
!false // true
```
